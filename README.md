## Setting up the project

```bash
# clone the project
git clone https://github.com/xcitic/Svelte-Githubination.git

# enter the project directory
cd Svelte-Githubination
yarn install
yarn run start

##### PROJECT WILL RUN ON #####
      http://localhost:5000
```
